# Plan ₿ Academy — Comprehensive Resource Guide

A curated companion reading list for each of the 41 courses in the Plan ₿ Academy curriculum. Resources are primarily **books, academic papers, BIPs, and core technical documents**, with a few foundational online docs where they remain the canonical reference. Each course is tiered:

- **Beginner / Foundational** — gentle, conceptual entry points.
- **Intermediate** — deeper technical or theoretical reading.
- **Advanced / Master** — primary sources, original papers, and reference manuals to reach mastery.

Read them in tier order. Within each tier, lighter items appear before denser ones.
