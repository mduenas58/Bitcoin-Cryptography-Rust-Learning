## 📖 Overview

Welcome to the **DevOps Mastery Specialization** hands-on labs repository! This comprehensive collection contains practical exercises and projects for all 8 courses in the DevOps Mastery Specialization. Whether you're just starting your DevOps journey or looking to enhance your skills, this repository provides step-by-step labs to help you master essential DevOps tools and practices.

### What You'll Learn
- **Infrastructure as Code (IaC)** with Terraform and Ansible
- **Containerization** using Docker
- **Container Orchestration** with Kubernetes
- **CI/CD Pipelines** using Jenkins
- **Version Control** with Git
- **Programming Fundamentals** with Python
- **Linux Administration** and Command Line
- **Microservices** Architecture
- **Configuration Management**

## 🔧 Prerequisites

- Basic computer literacy
- Laptop/desktop for practicing labs
- Enthusiasm to learn DevOps!
- No prior programming experience required (we start from scratch)

## 🚀 Getting Started

1. **Clone this repository:**
   ```bash
   git clone https://github.com/yourusername/devops-mastery-specialization.git
   cd devops-mastery-specialization
   ```

2. **Navigate to the course you're interested in:**
   ```bash
   cd course-01-devops-prerequisites
   ```

3. **Follow the lab instructions** in each course directory
4. **Complete the hands-on exercises** and projects
5. **Practice regularly** to reinforce your learning

## 💡 How to Use This Repository

- **For Beginners:** Start with Course 1 and progress sequentially
- **For Experienced Learners:** Jump directly to specific courses you need
- **For Practice:** Each lab includes exercises with increasing difficulty
- **For Projects:** Real-world scenarios to build your portfolio

### Lab Format
Each lab includes:
- Clear learning objectives
- Step-by-step instructions
- Code snippets and examples
- Validation checks
- Troubleshooting tips
